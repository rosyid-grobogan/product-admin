## Getting Started
- Download / Clone from github

- Set up your database in .env file

- Install composer
  ```
  composer install
  ```

- Generate Key
  ```
  php artisan key:generate
  ```
  
- run migration and seeding
  ```
  php artisan migrate:fresh --seed
  ```

- create new account  

- run NPM watch (if need)
  ```
  npm run watch
  ```
